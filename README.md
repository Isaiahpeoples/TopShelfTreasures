## 🍸 Project Overview

**Topshelf Treasures** is an interactive beverage discovery website that allows users to browse a wide variety of cocktails using the [CocktailDB API](https://www.thecocktaildb.com/), explore smooth content sliders via **Swiper.js**, and experience modern responsive design — all built with minimal external dependencies.

## 🚀 Technologies Used

| Technology    	| Description                                                             	|
|-------------------|-----------------------------------------------------------------------------|
| **HTML**      	| 🧱 Structures the website's content and layout.                         	|
| **CSS**       	| 🎨 Delivers a clean, modern, and responsive aesthetic.                  	|
| **JavaScript**	| ⚙️ Handles interactive elements and dynamic API-driven content.         	|
| **Swiper.js** 	| 📱 Enables sleek, swipeable sliders for better UX.                      	|
| **CocktailDB API**| 🍹 Provides cocktail data dynamically via JavaScript fetch requests.     	|
| **FormSpree** 	| 📬 Manages contact form submissions with ease.                          	|

<br/>

## 📑 Key Features

- 🌀 **Swiper.js Integration**: Smooth and responsive carousel sliders.
- 🍸 **CocktailDB API**: Real-time cocktail data and search functionality.
- 💬 **Form Handling**: Contact form powered by FormSpree.
- 📱 **Responsive Design**: Optimized for desktop, tablet, and mobile devices.
